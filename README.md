# exercice_regex
Two exercise for regex and unit test

# Pokemon

Get from the pokemon.html file a dictionnary of pokemon and their link such as :

```python
{
    "Abomasnow": "/wiki/Abomasnow",
    "Absol": "/wiki/Absol",
    "Accelgor": "/wiki/Accelgor"
    ...
}
```

Add a unit test to check if the data are the same as the one in pokemon.yaml

If you are fast enough, get the number of the pokemon and add it to the db such as :

```python
{
    "Abomasnow": {
        "link": "/wiki/Abomasnow",
        "number": "1"
        }
        ,
    ...
}
```

# Pollution

From two dataset, get the most polluted (MOYPTOT) station and get the name of the commune.

If you are fast enough, plot a heatmap of the pollution.
